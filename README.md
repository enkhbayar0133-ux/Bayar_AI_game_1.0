# Bayar_AI_game_1.0

## Engine Version
- Unreal Engine 5.3

## Required Plugins
- GameplayAbilities
- UMG

## Opening and Building in Unreal Editor
1. Open Unreal Engine 5.3 and select **Open Project**.
2. Navigate to the repository folder and open `Aura.uproject`.
3. Allow the editor to download or enable the required plugins if prompted, then restart the editor if necessary.
4. Once loaded, choose **Build** to compile the project, then click **Play** to run in-editor.
5. Use **File > Package Project** to create a distributable build for your target platform.
